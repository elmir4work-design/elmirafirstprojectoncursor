# elmirafirstprojectoncursor
This repository documents the setup process for installing and configuring Cursor AI along with the Claude Code and Codex extensions. The project includes the completed installation steps, extension setup, account login process, and notes about the workflow during the setup process.

— What Tools I Installed

* Cursor IDE
* Claude Code Extension
* Codex Extension
* GitHub Account and Public Repository Setup

— What Steps I Completed

* Researched how to download and use Cursor
* Learned the basics of GitHub and repository setup
* Installed Cursor successfully
* Installed and logged into the Claude Code extension
* Installed and logged into the Codex extension
* Created a GitHub account
* Created a public GitHub repository
* Created and edited a README.md file for documentation

—  What Issues I Ran Into and How I Solved Them

One of the main challenges was my limited familiarity with Cursor and GitHub, as I had little prior experience using these tools. I was also initially unsure about how the extensions and repository workflow functioned together.

To solve this, I conducted independent research by reading documentation, watching tutorials, and exploring the platforms directly. Through this process, I was able to better understand the installation process, extension setup, repository creation, and the overall workflow of the tools.
